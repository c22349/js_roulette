# ルーレット
<h2>言語</h2>
<li>JavaScript</li>

<h2>構成</h2>
<li>トップページ：roulette.html</li>

<h2>詳細</h2>
<li>スタートを押すランダムで動きます。</li>
<li>ストップを押すと数字に止まります。</li>
<li>スタートとストップを繰り返します。</li>
<li>全て揃うコメントが出てきます。</li>
<li>リセットを押して再度始めることができます。</li>
